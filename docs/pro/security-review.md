# PortBay Cloud — Security Review (moved)

The PortBay Cloud pre-launch security review lives in the **private
`portbay-cloud` repository** (`docs/security-review.md`), not here.

It enumerates backend security findings and hardening tasks (secret handling,
webhook verification, key rotation) that concern the proprietary Cloud service,
so it is kept out of the public AGPL repository to avoid telegraphing the private
implementation. See [repo boundaries](../architecture/repo-boundaries.md).

For the **Community app's** security policy — threat model, reporting, secret
scanning — see [`SECURITY.md`](../../SECURITY.md).
