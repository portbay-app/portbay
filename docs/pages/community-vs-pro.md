# Community vs. Pro

PortBay comes in two tiers. Community is the open-source core — free, local-first, and complete on its own. Pro and Cloud add optional features for teams and individuals who want more.

**Community is not a trial. Nothing in it is crippled to push you toward Pro.**

## How to Get Pro

Pro is not a subscription. It is a perpetual license tied to your GitHub account, earned one of two ways:

**1. Merged qualifying pull request**  
Contribute a meaningful improvement to PortBay Community and have it merged. What counts as qualifying is documented in [Contributing](./contributing.md). In general: code or documentation that genuinely improves the project for other users.

**2. Sponsorship**  
A qualifying donation via [GitHub Sponsors](https://github.com/sponsors/portbay-app) earns a perpetual Pro license. Sponsorships also directly fund ongoing development of the Community edition.

Both paths result in the same thing: a lifetime Pro license with no renewal required.

---

## A Note on Community's Independence

Community runs entirely offline. It does not require a PortBay account, an internet connection, or any Cloud infrastructure to function. Pro and Cloud features are opt-in additions — you connect them when and if they are useful to you.

If PortBay Cloud were to shut down tomorrow, Community would continue working exactly as it does today.
