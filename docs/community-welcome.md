**PortBay is an open-source, container-free, native local-development manager for
macOS.** It runs your local sites with trusted HTTPS and `.test` domains, handles
mixed Node / PHP / static / database stacks, and stays small and native — no
daemon zoo, no containers idling at gigabytes.

This is the place to:

- **Ask setup questions** — installs, runtimes, DNS/HTTPS, migrating from Herd /
  ServBay / MAMP.
- **Share what you're building** and how PortBay fits your workflow.
- **Discuss design and roadmap** before opening issues.

A few pointers:

- 🐛 **Bugs** → open an [issue](https://github.com/portbay-app/portbay/issues)
  with steps to reproduce.
- 🛠️ **Want to contribute?** Code contributions are open and a merged qualifying
  PR earns you [PortBay Pro](https://docs.portbay.app/pro/) — see
  [CONTRIBUTING](https://github.com/portbay-app/portbay/blob/main/CONTRIBUTING.md).
- 💙 **Support the project** by donating or contributing — PortBay stays
  Apache-2.0 either way.

PortBay is early (pre-1.0) and built in the open. Tell us what's rough — that
feedback is what shapes the roadmap.
