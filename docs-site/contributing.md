# Contributing

PortBay is not open to external contributions yet. The project is still in early validation and release-readiness work.

## Current Position

- Issues and pull requests are not the primary workflow yet.
- A full `CONTRIBUTING.md` will be added when the public development process is ready.
- The docs site will link to that file once it exists.

## Before Filing Feedback

When reporting behavior to the maintainers, include:

1. PortBay commit or build version.
2. macOS version.
3. Project type.
4. Error code from the `CommandError` envelope, if present.
5. Sidecar status.
6. Relevant project logs.
